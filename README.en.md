Activities for the **Data Structures and Algorithms II (EDA II)** course at the Faculty of Engineering, UNAM.

## 📘 What You'll Find Here:

- 💻 **In-class activities** in C++.
- 🧪 **Lab practice reports** (P1 – P11).
- 📝 **Reports** and documentation.
- 📚 **Assignments** for the semester.

## 🛠️ Technologies & Applied Concepts

- **C++** — Object-oriented programming, templates.
- **Advanced data structures:** Trees (BST, AVL), graphs, hash tables.
- **Advanced searching and sorting algorithms.**
- **Visual Studio** as the development environment.
